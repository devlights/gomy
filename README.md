# What's this?

This package is a personal library for Go language.

For more information, see https://pkg.go.dev/github.com/devlights/gomy .


## Requirements

### [go-task](https://taskfile.dev/)

```sh
$ go install github.com/go-task/task/v3/cmd/task@latest
```

## Howto

### Build

```sh
$ task fmt vet
$ task build
```

### Test

```sh
$ task test
```

### Cover

```sh
$ task cover
```
